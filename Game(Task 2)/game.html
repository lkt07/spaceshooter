SPACE SHOOTER

Controls

  • A (or) Left Arrow Key - Moves the space cannon towards the left
  • D (or) Right Arrow Key - Moves the space cannon towards the Right

About Space Shooter

  • Each rock will have a number embedded in it. That many bullets are required to take it down
  • The bullets will be generated from the cannon automatically
  • Use the controls to navigate the space cannon which in turn allows you to control the bullets generated
  • The only way the game gets over will be when the rocks hit the space cannon. So try to destroy as many rocks as possible to rack up a new highscore everytime you play.
